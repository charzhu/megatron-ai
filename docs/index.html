      Quick start
      
Boot the swarm in three steps

      
Minimal ceremony. Maximum menace.

    

    

      

        
1Initialize

        
npx -y github:charzhu/megatron-ai init

      

      

        
2Configure MCP client

        
{
  "servers": {
    "megatron-swarm": {
      "type": "stdio",
      "command": "node",
      "args": ["${workspaceFolder}/megatron-plugin/dist/mcp-server.js"],
      "env": {
        "MEGATRON_WORKSPACE_ROOT": "${workspaceFolder}",
        "DOTENV_PATH": "${workspaceFolder}/.env",
        "PATH": "${env:PATH}"
      }
    }
  },
  "inputs": []
}

      

      

        
3Start orchestrating

        
"Build a user authentication system with issue-first workflow,
deploy architect + developer + QA, and return a review-ready PR."
